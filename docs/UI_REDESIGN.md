# RockyFit UI Redesign

## Design Direction
- Cleaner, calmer, more premium.
- Less dense dashboard feel.
- One strong accent color only.
- Bigger typography, fewer borders, clearer hierarchy.

## Primary Screens

### Home / Today
- Today's workout card
- readiness card
- last session summary
- one primary CTA: Start Workout

### Workout
- large exercise cards
- sets logging first-class
- rest timer persistent and obvious
- substitution button per exercise
- end-of-session notes and rating

### Progress
- adherence streak
- weekly volume trend
- PR highlights
- recovery trend

### Coaching
- latest daily report
- archive of prior reports
- "what to focus on next" section

## UX Fixes
- reduce clutter above the fold
- stop making users hunt for the next action
- make generated program feel custom and owned by the user
- surface why the workout was built this way
