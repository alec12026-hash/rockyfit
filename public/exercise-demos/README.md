# Exercise Demo Assets

Drop normalized assets here:
- `exercise-id.mp4` (required)
- `exercise-id.jpg` (poster, required)

Examples:
- `bench_press.mp4`
- `bench_press.jpg`
- `deadlift.mp4`
- `deadlift.jpg`

The app auto-loads these from `lib/exerciseInfo.ts` using:
`/exercise-demos/<exercise_id>.mp4`
